## Consulting Chart Add-ins

### think-cell
**The gold standard for consulting charts**

| Attribute | Detail |
|-----------|--------|
| **Best for** | MBB consultants, investment banks, corporate strategy |
| **Key features** | Waterfall charts, Mekko/Marimekko, Gantt, automatic agendas, Excel linking |
| **Pricing** | ~$299/year per user |
| **Platform** | Windows only (Mac support limited) |
| **Verdict** | Industry standard at top consulting firms. Worth it if you build complex data visualizations daily. |

**Pros:**
- Unmatched chart quality and control
- Automatic agenda and table of contents
- Deep Excel integration
- Industry-standard at MBB

**Cons:**
- Expensive
- Windows-centric (Mac support is a work-in-progress)
- Steep learning curve

---

### Deckary
**Modern think-cell alternative at 1/3 the price**

| Attribute | Detail |
|-----------|--------|
| **Best for** | Boutique consultants, Mac users, budget-conscious teams |
| **Key features** | Waterfall, Mekko, Harvey balls, AI slide generation, productivity tools |
| **Pricing** | $49-119/year |
| **Platform** | Windows + Mac (full parity) |
| **Verdict** | Best value for consultants who need think-cell-style charts occasionally. Full Mac support is a differentiator. |

**Pros:**
- Significantly cheaper than think-cell
- Full Mac support
- Includes AI slide generation
- Modern interface

**Cons:**
- Smaller user base
- Fewer advanced chart customization options
- Newer product (less battle-tested)

---

### Ampler Charts

| Attribute | Detail |

| **Platform** | Windows + Mac |

## AI Presentation Makers

### Gamma
**AI-first presentation creation**

| Attribute | Detail |
|-----------|--------|
| **Best for** | Fast first drafts, internal presentations, non-linear storytelling |
| **Key features** | Text-to-slides, interactive cards, web publishing, AI image generation |
| **Pricing** | Free tier available. Pro: $10-20/month |
| **Platform** | Web-based |
| **Verdict** | Best for speed. Not for final client deliverables (limited customization). Great for internal decks and brainstorming. |

**Pros:**
- Extremely fast deck generation
- Good for interactive web presentations
- Free tier is generous

**Cons:**
- Limited to Gamma's design system
- Not suitable for highly customized client work
- PowerPoint export quality varies

## Recommendations by Use Case

### For Boutique Consultants
1. **Primary:** Deckary ($49-119/year) — Best value for consulting charts
2. **Productivity:** Power-User ($50 one-time) — Comprehensive toolkit
3. **Images:** Pexels (free) — Stock photos

**Total annual cost:** ~$100-170

### For Ex-MBB Consultants
1. **Charts:** think-cell ($299/year) if budget allows, Ampler (~$99/year) if not
2. **Productivity:** PPT Productivity ($99/year) — Replaces firm tools
3. **AI assist:** Gamma (free tier) — Fast first drafts

**Total annual cost:** ~$200-400

### For Corporate Strategy Teams
1. **Charts:** think-cell ($299/year) — Industry standard
2. **Brand:** Efficient Elements (custom) — Design consistency
3. **Interactive:** Mentimeter ($12/month) — Meeting engagement

**Total annual cost:** ~$450-600

### For Trainers & Facilitators
1. **Interactive:** Mentimeter ($12/month) — Audience engagement
2. **Quizzes:** ClassPoint ($8/month) — In-slide gamification
3. **eLearning:** iSpring Free — Course publishing

**Total annual cost:** ~$240

### For Speed (Internal Decks)
1. **AI Generation:** Gamma (free tier) — Fastest deck creation
2. **Polish:** Beautiful.ai ($12.50/month) — Clean up designs
3. **Images:** Pexels (free)

**Total annual cost:** ~$150
