# Feature: Template Library — Visual Themes × Narrative Structures

**Status:** Planning  
**Type:** Feature Requirement (not a WOW)  
**Date:** 2026-02-14  

---

## Overview

A template library with two dimensions — users pick both a **visual theme** (how it looks) and a **narrative structure** (how it thinks). This is table stakes for any presentation tool, but the narrative structure dimension is a differentiator.

## Visual Themes (the look)

- Clean Corporate
- Bold Modern
- Minimal
- Classic Consulting
- User's own brand (extracted via WOW #2 — Deck Intelligence)

## Narrative Structures (the thinking)

Pre-built consulting-grade slide flows with guidance on what belongs in each slide and why:

- Strategy Recommendation
- Client Proposal / Pitch
- Board Update
- Quarterly Business Review (QBR)
- Market Analysis
- Implementation Plan
- Project Kickoff
- Competitive Analysis
- Business Case / Investment Thesis
- Change Management

Each structure includes:
- Pre-defined slide order with SCQA framing
- Per-slide guidance: "This slide should contain..." and "Why this matters here..."
- Recommended slide count and pacing
- Common variations (short version, detailed version)

## Differentiator

Most tools: "Here's a blue template with nice fonts"  
SlideHeroes: "Here's a Strategy Recommendation with pre-built SCQA flow, executive summary, situation analysis, options evaluation, recommendation, and implementation roadmap"

Competitors separate content from design. SlideHeroes marries them.

## Dependencies

- WOW #2 (Deck Intelligence) provides user's own brand templates
- Narrative structures feed into the Assemble → Outline flow
