# Slide Design Principles

## Overview

Effective slide design combines visual communication principles with clarity of message. Good slides amplify your spoken words without overwhelming your audience with text.

## Core Principles

### 1. One Idea Per Slide
Each slide should convey a single, clear concept. This creates cognitive ease for your audience and makes your message memorable.

**Instead of**: Cramming 5 different points on one slide
**Do**: Create 5 slides with one point each

### 2. The 10/20/30 Rule (Guy Kawasaki)
- 10 slides maximum
- 20 minutes maximum
- 30-point minimum font size

### 3. White Space is Your Friend
Empty space isn't wasted space—it directs attention and prevents visual overwhelm.

### 4. Consistency Builds Professionalism
Use consistent:
- Fonts (limit to 2 per presentation)
- Colors (brand palette + accent colors)
- Layout structures
- Spacing and alignment

## Typography Guidelines

### Font Selection
- **Sans-serif fonts** are generally easier to read on screens
  - Examples: Helvetica, Arial, Open Sans, Roboto
- **Limit to 2 fonts** maximum per presentation
- **Avoid decorative fonts** for body text

### Font Sizing
- **Titles**: 36-44pt minimum
- **Body text**: 24-32pt minimum
- **Captions/footnotes**: 18-20pt minimum

### Text Hierarchy
Use size and weight to create hierarchy:
- Title (largest, bold)
- Subtitle (medium, regular)
- Body (regular size, regular weight)
- Captions (smallest, lighter color)

## Color Principles

### Color Palette
- **Primary colors**: 1-2 brand colors
- **Secondary colors**: 2-3 supporting colors
- **Accent color**: 1 color for emphasis/calls-to-action
- **Neutral colors**: Black, white, grays for text and backgrounds

### Contrast
- Ensure sufficient contrast between text and background
- Dark text on light backgrounds OR light text on dark backgrounds
- Minimum contrast ratio: 4.5:1 for body text

### Color Psychology
- **Blue**: Trust, stability, professionalism
- **Green**: Growth, success, money
- **Red**: Urgency, excitement, importance
- **Orange**: Energy, enthusiasm
- **Purple**: Creativity, luxury

## Data Visualization

### Choosing the Right Chart

| Data Type | Best Chart |
|-----------|------------|
| Comparison over time | Line chart |
| Part-to-whole | Pie chart, stacked bar |
| Comparison between categories | Bar chart |
| Distribution | Histogram, scatter plot |
| Relationship between two variables | Scatter plot |
| Flow/process | Flowchart, Sankey diagram |

### Chart Best Practices
1. **Simplify**: Remove chart junk (unnecessary gridlines, borders, backgrounds)
2. **Label directly**: Don't rely solely on legends
3. **Start axis at zero** (unless using index charts)
4. **Use color purposefully**: Highlight the key data point
5. **Maximum 3-4 data series** per chart

### Before/After Example

**Before** (cluttered):
- 3D pie chart with 8 segments
- Light blue background
- No labels on segments
- Legend in corner

**After** (clean):
- Horizontal bar chart with 4 segments
- White background
- Direct labeling
- Accent color on most important segment

## Layout Principles

### The Rule of Thirds
Divide your slide into a 3x3 grid. Place key elements along the lines or at intersections.

### Visual Hierarchy
1. **Primary**: Where does the eye go first? (usually top-left or center)
2. **Secondary**: Where next?
3. **Tertiary**: Supporting details

### Alignment
- Use consistent left, right, or center alignment
- Align elements to each other
- Use grids to maintain consistency

### Common Layouts
- **Title + body**: Simple, effective
- **Two columns**: Comparison or并列
- **Image + text**: Visual + explanation
- **Centered**: Emphasis on single element

## Image Guidelines

### Image Quality
- Use high-resolution images (minimum 1920x1080 for full-bleed)
- Avoid pixelated or stretched images
- Use consistent image styles (photos OR illustrations, not both)

### Image Placement
- Leave margins around images
- Don't crowd images with text
- Use consistent image sizing across slides

### Stock Photo Tips
- Avoid generic, overused stock photos
- Look for authentic, specific imagery
- Consider illustrations over photos for abstract concepts

## Animation and Transitions

### Use Sparingly
- Keep animations simple (fade, slide)
- Avoid animation that draws attention away from content
- Rehearse timing

### Best Practices
- Use consistent transitions between slides
- Animate elements to appear in reading order (left-to-right, top-to-bottom)
- Keep builds simple—no more than 3-4 steps per slide

## Accessibility

### Font and Contrast
- Minimum 24pt for body text
- Contrast ratio of at least 4.5:1
- Avoid red/green combinations (color blindness)

### Structure
- Reading order should match visual order
- Use built-in slide layouts (not just positioned text boxes)
- Add alt text to images and charts

### Distribution
- Provide accessible versions when sharing
- Don't rely solely on color to convey meaning
- Use patterns/shapes in charts in addition to colors

## Presentation-Specific Tips

### For Projectors
- Test on the actual display beforehand
- Use high contrast
- Avoid subtle colors
- Make text larger than you think necessary

### For Screens
- Dark mode can reduce eye strain
- Account for different aspect ratios
- Test on multiple devices

### For Print
- Use sufficient contrast
- Add page numbers
- Consider landscape vs portrait

## Quick Checklist

Before finalizing slides:
- [ ] Can I read all text from the back of the room?
- [ ] Is there one main idea per slide?
- [ ] Is there consistent alignment?
- [ ] Are fonts consistent?
- [ ] Do charts have clear labels?
- [ ] Is there enough white space?
- [ ] Do colors provide sufficient contrast?
