The `checkHydration` timeout at line 117:32 in `form-submission-protection.ts` is not being properly cleaned up when tests complete. This suggests:

1. A `setTimeout` or `setInterval` is created during test execution

- Test suite should exit cleanly with no pending asynchronous operations

2. **Add proper cleanup**:
