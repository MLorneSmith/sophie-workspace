## 🐛 Technical Issue

The `TestBillingGateway.createTestSubscription()` method attempts to create subscription records but fails due to RLS permissions:

```
Failed to create test subscription: {
  code: '42501',
  message: 'permission denied for function upsert_subscription'
}
```

**Root Cause:** The `upsert_subscription` RPC function requires `service_role` permissions, but the test environment uses authenticated user permissions.
