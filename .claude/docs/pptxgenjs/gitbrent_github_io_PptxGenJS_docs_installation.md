
```codeBlockLines_e6Vv

```

### Using CDN [​](https://gitbrent.github.io/PptxGenJS/docs/installation/\#using-cdn "Direct link to heading")

[jsDelivr Home](https://www.jsdelivr.com/package/npm/pptxgenjs)

Bundle: Modern Browsers and IE11

```codeBlockLines_e6Vv
<script src="https://cdn.jsdelivr.net/npm/pptxgenjs@3.12.0/dist/pptxgen.bundle.js"></script>

```

Min files: Modern Browsers

```codeBlockLines_e6Vv
<script src="https://cdn.jsdelivr.net/npm/pptxgenjs@3.12.0/libs/jszip.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/pptxgenjs@3.12.0/dist/pptxgen.min.js"></script>

```
