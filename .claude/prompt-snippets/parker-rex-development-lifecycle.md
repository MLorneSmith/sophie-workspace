# 1. Problem ID

Problem Definition & Market Validation

## What it covers

Identifying real user problems, validating demand, defining success metrics

## Role in ecosystem

Foundation that prevents building the wrong thing; defines all downstream decisions

## What happens without it

Build features nobody wants, waste months of development time, poor product-market fit

## Tools

User interviews, surveys, analytics tools, competitor analysis, landing page tests

## Job titles

Product Manager, Product Owner, Founder, UX Researcher

## Resources

"The Mom Test" book, Y Combinator Startup School, Jobs-to-be-Done framework
