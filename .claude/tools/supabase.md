# Supabase MCP Server

> Connect your Supabase projects to Cursor, Claude, Windsurf, and other AI assistants.

[Full README content as shown in the original GitHub page]

The README provides a comprehensive guide to setting up and using the Supabase Model Context Protocol (MCP) Server, which allows AI assistants to interact with Supabase projects. Key highlights include:

1. Prerequisites: Requires Node.js installation
2. Setup process:

   - Create a personal access token
   - Configure MCP client
   - Optional project scoping and read-only modes

3. Detailed list of available tools across categories:
   - Project Management
   - Database Operations
   - Edge Function Management
   - Project Configuration
   - Branching
   - Development Tools
   - Cost Confirmation

The document emphasizes that this is a pre-1.0 server, so breaking changes might occur. It's licensed under Apache 2.0 and aims to standardize how Large Language Models interact with Supabase services.
