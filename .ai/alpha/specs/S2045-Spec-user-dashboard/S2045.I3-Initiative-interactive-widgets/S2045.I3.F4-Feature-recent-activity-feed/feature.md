## Description
Build a vertical timeline-style Activity Feed component for Row 2 of the dashboard that displays the 10 most recent user activity events from the `activity_events` table. Each event shows a type-specific icon, description, and relative timestamp (e.g., "Today, 2:30 PM", "Yesterday", "Feb 7"). Events are generated by database triggers in I1 and include lesson completions, quiz scores, assessment completions, and presentation creations.

## User Story
**As a** SlideHeroes user
**I want to** see a timeline of my recent learning activities on my dashboard
**So that** I can track my progress and remember what I've accomplished recently

## Architecture Decision

**Approach**: Pragmatic
**Rationale**: The timeline UI is the only truly new visual component in this initiative. Use a simple CSS-based vertical timeline (border-left line with absolute-positioned dots) rather than a heavy timeline library. The component is a Server Component receiving pre-fetched events as props.

### Key Architectural Choices
1. Server Component receives activity events array from dashboard loader
2. Timeline UI built with Tailwind CSS (border-left, relative positioning for dots)
3. Event type → icon mapping in a shared utility (reusable if activity feed appears elsewhere)
4. Relative time formatting using `Intl.DateTimeFormat` + custom logic (no `date-fns` dependency needed)

### Trade-offs Accepted
- No real-time updates (data refreshes on page load only, per spec out-of-scope)
- Fixed limit of 10 events (no pagination or "load more" in v1)
- Timeline UI is custom CSS rather than a component library (more control, slightly more code)

### Component Strategy

| UI Element | Component | Source | Rationale |
|------------|-----------|--------|-----------|
| Card wrapper | Card, CardHeader, CardTitle, CardContent | @kit/ui/card | Standard widget pattern |
| Event icons | Book, Trophy, ClipboardCheck, FileText | lucide-react | Type-specific visual cues |
| Timeline dots | Custom div with `rounded-full` | Tailwind CSS | No existing timeline component in shadcn |
| Timeline line | `border-l-2` on container | Tailwind CSS | Simple vertical line |
