**Agents available from Outline onward** as a persistent side panel, not a separate step. Coach/Skeptic work at Outline level (narrative feedback), not just post-generation.

**Templates** selected at Generate step for visual output.

## [16:45] GPT Coding Performance — Context Engineering Research
Mike asked how to maximize GPT sub-agent coding performance. Research findings:

**Key insight from ETH Zurich study (2602.11988, Feb 2026):**
- LLM-generated AGENTS.md files **reduced** success rates in 5/8 tests
- Human-written files only improved by ~4% on average
- Both increased inference costs by 20%+ (more reasoning tokens wasted)
- Recommendation: **minimal, hand-written context files** with only genuinely non-discoverable info

**Current state:** SlideHeroes repo has AGENTS.md (293 lines) + CLAUDE.md (913 lines) — likely too long per research.

**Action items discussed:**
1. Trim AGENTS.md to ~50-80 lines of essential non-discoverable info
2. Put task-specific context in spawn prompts instead of giant instruction files
3. Consider Context7 skill for framework docs (proven effective by Vercel for framework-specific knowledge)
4. GPT-5.3 Codex is 25% faster with interactive steering — worth tracking for API availability
