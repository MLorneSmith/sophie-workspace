## [16:40] Codex Sub-Agent Debugging — IMPORTANT LESSON

   - Repo `.codex/skills/` (committed symlinks)
3. **18 non-coding skills** loaded as context for every task (seo-audit, canvas-design, pptx, etc.)
4. **Repo `.codex/skills/`** has symlinks committed to git pointing to `../../.agents/skills/` — these break in worktrees

**Fixes applied:**
- Renamed `~/.agents/skills/brainstorming` → `~/.agents/skills/.brainstorming-disabled`
- Removed brainstorming symlink from `~/.codex/skills/`
- Trimmed `~/.codex/skills/` to 5 coding-relevant skills: context7, frontend-design, supabase-postgres-best-practices, tailwind-design-system, vercel-react-best-practices
