## Key Decisions

### Todoist Dual-Date Strategy
**Pattern:** Parent task due date = hard deadline (syncs to MC `deadline`). Todoist sub-tasks = scheduled work sessions (show in Today view).

**SOP v2:** Todoist has native `deadline` field (added Jan 2025). Use it instead of sub-task workarounds.
- MC `deadline` → Todoist `deadline` field
- Todoist `due` = first planned work day
- Sub-tasks only for large tasks with 3+ distinct phases

### SOP Tagging Strategy

- Cloudflare Turnstile blocks ALL headless browsers at engine level
