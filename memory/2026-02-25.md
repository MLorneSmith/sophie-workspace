## [21:30] Credit Burn Crisis - Root Cause Analysis

**Problem:** Both ChatGPT Pro and Claude Max hit usage limits this week.

**Root causes identified:**
1. **Sub-agent Watchdog** — 288 Claude calls/day (every 5 min)
2. **MC ↔ Todoist Sync** — 48 GPT-5 calls/day (every 30 min)
3. **Nightly coding sessions** — ~100 GPT-5 calls/night combined
4. **Morning Brief** — ~20 GPT-5 calls/day
5. Multiple other cron jobs using LLMs for deterministic tasks

**Key insight:** LLMs were being used as glorified `if/then/else` statements for tasks that could be deterministic Python scripts.

## [23:30] GPT-5.2 vs GLM-5 for Coding Sub-Agents

**Finding:** GPT-5.2 via ccproxy gets stuck in "thinking" loops without executing tools. GLM-5 executes cleanly.

**Evidence:**
- Task #485: GPT-5.2 looped on same git/find commands 10+ times
- GLM-5: Explored → understood → created files → committed → pushed → created PR

**Decision:** Use `zai/glm-5` for coding sub-agents. Keep GPT-5.2 for interactive/local use only.

**Updated:** `~/clawd/.ai/agents/coder.yaml` — model: `zai/glm-5`
