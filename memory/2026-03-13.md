**Fixes:**
- PR #488 (merged): Hardened auto-deploy script — `git reset --hard origin/main` instead of ff-only, kill zombie processes before restart, fixed pm2 app name, added health check
- Disabled + deleted systemd service (`internal-tools.service`) — pm2 is sole process manager now
- **Learning:** systemd and pm2 should never manage the same port
