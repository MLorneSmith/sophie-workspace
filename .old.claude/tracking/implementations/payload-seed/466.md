# Task: Build seed orchestrator

## Description

Implement the main seed orchestrator that coordinates the entire seeding process - processing collections in dependency order, managing the reference cache lifecycle, tracking progress, and handling errors. This is the central coordinator for all seeding operations.

## Acceptance Criteria

- [ ] Orchestrator processes collections in correct dependency order
- [ ] Reference cache lifecycle managed properly
- [ ] Progress tracking integrated throughout
- [ ] Error handling and retries working
- [ ] Dry-run mode implemented (validation only)
- [ ] Collection filtering working (--collections flag)
- [ ] Post-seed validation implemented
- [ ] Integration tests passing

## Technical Details

### Implementation Approach

1. Create `seed-orchestrator.ts` with main workflow:

   ```
   1. Initialize Payload
   2. Load JSON data
   3. Validate references
   4. For each collection in SEED_ORDER:
      a. Pre-process records
      b. For each record:
         - Resolve references
         - Create via Payload API
         - Register in cache
      c. Post-process results
   5. Report summary
   6. Cleanup
   ```

2. Implement collection ordering from config
3. Add reference cache management
4. Integrate progress tracking
5. Implement dry-run mode
6. Add post-seed validation

### Files to Create

- `apps/payload/src/seed/seed-engine/core/seed-orchestrator.ts` - Main orchestrator

### Workflow

```typescript
export class SeedOrchestrator {
  async run(options: SeedOptions): Promise<SeedResult> {
    // 1. Initialize
    const payload = await initializePayload();
    const resolver = new ReferenceResolver();
    const progress = new ProgressTracker();

    // 2. Load data
    const data = await loader.loadAllCollections();

    // 3. Validate
    await this.validateReferences(data, resolver);

    // 4. Process collections
    for (const collection of SEED_ORDER) {
      await this.processCollection(collection, data, resolver, progress);
    }

    // 5. Verify
    await this.postSeedValidation();

    return result;
  }
}
```
