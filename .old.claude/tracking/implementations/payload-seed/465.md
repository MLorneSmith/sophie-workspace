# Task: Implement progress tracker utility

## Description

Create the progress tracking utility that displays real-time progress bars, collection status, and summary reports during seeding operations. This provides user feedback and makes the seeding process transparent.

## Acceptance Criteria

- [ ] Progress bar implementation with ASCII characters
- [ ] Collection reporting with success/failure counts
- [ ] Summary generation with timing and throughput metrics
- [ ] ETA calculation for remaining work
- [ ] Colored output using chalk
- [ ] Real-time updates without flickering
- [ ] Unit tests passing with >85% coverage

## Technical Details

### Implementation Approach

1. Create `progress-tracker.ts` with:
   - `startCollection(name, total)` - Begin tracking collection
   - `updateProgress(current, total)` - Update progress bar
   - `completeCollection(stats)` - Finish collection with stats
   - `generateSummary()` - Create final summary report
2. Implement progress bar rendering:
   - Use ASCII characters: `████████████████████`
   - Show percentage: `[████████████████████] 25/25 (100%)`
   - Update in place without creating new lines
3. Add performance tracking:
   - Track start/end times per collection
   - Calculate records per second
   - Identify slowest collections

### Files to Create

- `apps/payload/src/seed/seed-engine/utils/progress-tracker.ts` - Progress tracking
- `apps/payload/src/seed/seed-engine/utils/progress-tracker.test.ts` - Unit tests
