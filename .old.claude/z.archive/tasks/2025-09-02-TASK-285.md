## 📋 Overview

Implement a devcontainer configuration to enable Claude Code integration with GitHub Codespaces, allowing developers to work in a fully configured cloud development environment while maintaining compatibility with local development setups.

## 🎯 Background & Motivation

### Current State

- **Hybrid Architecture**: Next.js runs on host machine, Supabase and MCP servers run in Docker containers
- **Local-only Development**: Currently requires local machine setup with specific dependencies
- **Claude Code Integration**: Works locally but lacks cloud development support

### Why This Matters

1. **Remote Development**: Enable development from any device with browser access
2. **Consistent Environment**: Eliminate "works on my machine" issues
3. **Faster Onboarding**: New developers can start coding in minutes, not hours
4. **Claude Code Cloud Support**: Full AI assistant capabilities in cloud environments
5. **Reduced Setup Complexity**: No local dependency management required

## 🔧 Technical Requirements

### Container Specifications

```json
{
  "image": "mcr.microsoft.com/devcontainers/typescript-node:20",
  "features": {
    "ghcr.io/devcontainers/features/docker-in-docker:2": {},
    "ghcr.io/devcontainers/features/github-cli:1": {},
    "ghcr.io/devcontainers/features/node:1": {
      "version": "20"
    }
  },
  "customizations": {
    "vscode": {
      "extensions": [
        "Anthropic.claude-code",
        "dbaeumer.vscode-eslint",
        "esbenp.prettier-vscode",
        "Prisma.prisma",
        "bradlc.vscode-tailwindcss"
      ]
    }
  }
}
```

### Service Architecture

- **Primary Container**: Node.js development environment
- **Supabase Services**: PostgreSQL, Auth, Storage, Realtime
- **MCP Servers**: Running as background processes
- **Port Forwarding**: 3000 (Next.js), 54321 (Supabase Studio), 54322 (PostgreSQL)

### Resource Requirements

- **Minimum**: 4 cores, 8GB RAM
- **Recommended**: 8 cores, 16GB RAM
- **Storage**: 32GB minimum for container and dependencies
