## Proposed Solution Architecture

### Docker Compose Setup

Create a `docker-compose.mcp.yml` file with services for each MCP server:

```yaml
version: '3.8'
services:
  mcp-perplexity:
    build: ./.mcp-servers/perplexity
    environment:
      - PERPLEXITY_API_KEY=${PERPLEXITY_API_KEY}
    ports:
      - '3001:3000'

  mcp-supabase:
    build: ./.mcp-servers/supabase
    environment:
      - SUPABASE_ACCESS_TOKEN=${SUPABASE_ACCESS_TOKEN}
    ports:
      - '3002:3000'
  # ... additional services
```

### Individual Dockerfiles

Each server gets its own optimized container:

```dockerfile
# Example: .mcp-servers/perplexity/Dockerfile
FROM node:22-alpine
WORKDIR /app
RUN npm install -g server-perplexity-ask
EXPOSE 3000
CMD ["server-perplexity-ask"]
```

### Updated .mcp.json Configuration

Replace npx commands with HTTP connections to local containers or direct executables:

```json
{
  "mcpServers": {
    "perplexity-ask": {
      "command": "docker",
      "args": ["exec", "mcp-perplexity", "server-perplexity-ask"]
    }
  }
}
```
