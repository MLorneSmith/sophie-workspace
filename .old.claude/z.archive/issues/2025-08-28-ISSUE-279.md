## Root Cause Analysis

While Issue #277's fix correctly added `detached: true` to server processes, **3 separate cleanup routines** are still actively killing development servers:

### Problematic Code Patterns (.claude/scripts/test-controller.cjs)

2. **Line 1247** (Recovery): `pkill -9 -f "pnpm.*dev:test"`  
3. **Line 1649** (Post-test): `pkill -f "dev:test"`

These patterns **exactly match** our server commands:

- `pnpm --filter web dev:test`
- `pnpm --filter payload dev:test`

**The `detached: true` fix prevents signal propagation but cannot prevent direct process killing by pattern matching.**
