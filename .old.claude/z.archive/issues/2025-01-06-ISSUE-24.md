## Initial Analysis

This appears to be a pnpm workspace dependency resolution issue where:

1. Vitest is installed only at the root level
2. The web app workspace expects it locally in `apps/web/node_modules/`
3. pnpm is not hoisting or linking the dependency properly
4. This blocks all unit test execution in the web app

## Suggested Investigation Areas

1. **Dependency Configuration**: Add vitest as explicit devDependency to web app package.json
2. **Workspace Configuration**: Check pnpm-workspace.yaml for dependency hoisting settings
3. **Package Manager**: Verify pnpm installation and workspace setup
4. **Alternative Approach**: Run tests from root level instead of workspace level
