# Resend SMTP Production Configuration Guide

**Reference**: Issue #243 - Implement Resend SMTP for Supabase Email Delivery  
**Documentation**: EMAIL-CONFIGURATION-GUIDE.md

## Production Environment Variables

Set these in your deployment platform (Vercel, Railway, etc.) - **DO NOT commit actual values**:

```env
# Set this to 'resend' to enable Resend email provider
MAILER_PROVIDER=resend

# Your Resend API key (format: re_xxxxxxxxxxxxxxxxxxxxxxxx)
# Generate this from: https://resend.com/api-keys
RESEND_API_KEY=re_your_actual_api_key_here

# Email sender configuration
# Use your verified domain from Resend
EMAIL_SENDER="SlideHeroes <noreply@slideheroes.com>"
CONTACT_EMAIL=contact@slideheroes.com
```

## Supabase SMTP Settings Reference

Configure these in **Supabase Dashboard → Authentication → Settings**:

| Setting | Value |
|---------|-------|
| SMTP Host | `smtp.resend.com` |
| SMTP Port | `465` (SSL/TLS) |
| SMTP Username | `resend` |
| SMTP Password | [Use the same RESEND_API_KEY value] |
| Sender Email | `noreply@slideheroes.com` |
| Sender Name | `SlideHeroes` |

## DNS Records Checklist

Add these DNS records to your domain:

### SPF Record

```
Name: @
Type: TXT
Value: v=spf1 include:resend.com ~all
```

### DKIM Record (from Resend dashboard)

```
Name: resend._domainkey
Type: TXT
Value: [Generated by Resend after domain verification]
```

### DMARC Record

```
Name: _dmarc
Type: TXT
Value: v=DMARC1; p=quarantine; pct=100; rua=mailto:dmarcreports@slideheroes.com
```

## Testing Checklist

After configuration, test these flows:

- [ ] User signup (should receive confirmation email)
- [ ] Password reset (should receive reset link)
- [ ] Email deliverability check using mail-tester.com
- [ ] Verify emails reach inbox, not spam folder
- [ ] Check Supabase Auth logs for any SMTP errors

## Monitoring

Monitor email delivery through:

- **Resend Dashboard**: <https://resend.com/emails>
- **Supabase Auth logs**: Project → Logs → Auth
- **Email deliverability tools**: mail-tester.com, mxtoolbox.com

## Implementation Timeline

1. **Phase 1**: Resend account setup (~15 minutes)
2. **Phase 2**: DNS configuration (~30 minutes + 24-48 hours propagation)
3. **Phase 3**: Environment variables (~5 minutes)
4. **Phase 4**: Supabase SMTP configuration (~10 minutes)
5. **Phase 5**: Testing & validation (~15 minutes)

**Total active work**: ~1-2 hours + DNS propagation time
